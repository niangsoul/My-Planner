# 🔑 핵심기능

### 1. 통합 일정 관리

> 하나의 달력 화면에서  
> 일정(Schedule) · 할 일(Todo) · 일기(Diary)를 한 번에 관리할 수 있습니다.

---

### 2. 습관 트래커

> 사용자가 설정한 습관을 기준으로  
> 일일 수행 여부를 체크하고 흐름을 관리할 수 있습니다.

---

### 3. 감정 기반 일기 기록

> 일기 작성 시 감정 상태를 함께 기록하여  
> 감정 흐름을 데이터로 저장하고 관리합니다.

---

### 4. 통계 시각화

> 일정 등록량, 할 일 완료율, 습관 실천율을  
> 차트로 시각화하여 사용자 활동을 한눈에 확인할 수 있습니다.
